Scalable layout

Mozaïk dashboards support multiple devices through adaptive layout. You can put a big screen in the open space or consult your dashboard on your smartphone.

Extendable by modules

Mozaïk widgets are maintained as separate modules, thus available via mozaik-ext-<name> in npm.js. more

Grid positioning

Mozaïk provides a simple way to define dashboard layout using a grid system. more

Optimized backend communication

Most extensions need to communicate with APIs, Mozaïk ease this by providing a backend which handles API calls and push data to widgets through websocket.
